# BigTSQuery

Run TypeScript AST Queries across all public TypeScript code on GitHub.

## Development

Starting dev server (http://localhost:4200/):

    yarn start

Starting storybook (http://localhost:4201/)

    yarn storybook

Running tests:

    yarn test
